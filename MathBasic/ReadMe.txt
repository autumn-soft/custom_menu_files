● インストール
~~~~~~~~~~~~~~~
MathBasic.zipに含まれているファイルは以下のとおりです。

ReadMe.txt : このファイル
MathBasic.txt : メニューファイルの本体

・以下のファイルはMathBasic.txtの中から自動的にインポートします。

MathTemplate.txt : ユーザーが自由に数式を登録するためのメニューファイル

このうち、「MathBasic.txt」と「MathTemplate.txt」の２つを
ペースターのデータフォルダにコピーして下さい。

そして、［ペースターの設定ダイアログ］−［カスタムメニュー］タブ
にある「カスタムメニューの設定」の「追加」ボタンを押して、
「MathBasic.txt」をリストに追加して下さい。
「MathTemplate.txt」の方は登録する必要はありません。

最後に、下にあるショートカットキーの設定の欄で、
お好きな呼び出しキーを選んで「設定▲」ボタンを押します。
私は、[Alt]+[Ctrl]+[E]キーを設定しています。

[Ctrl]+[Alt]+[M]キーは、ペースターのメニュー項目作成用の
カスタムメニューで昔から使用しているため、できれば避けてください。

● 使い方
~~~~~~~~~
「MathBasic.txt」の方には、LaTeXの基本的な数式コマンドが登録されています。
「MathTemplate.txt」の方には、ユーザーの方がよく使用する数式の
テンプレートなどを登録してみてください。
デフォルトでいくつかの数式を登録していますが、これらはサンプルですので
削除してしまって構いません。
また、バージョンアップを行う際にも「MathTemplate.txt」の方は変更しません。
